Debbynote's Markdown Syntax Guide

Text formatting:

  • **bold** → Bold text
  • *italic* → Italic text
  • ~~strikethrough~~ → Strikethrough text
  • [link](https://example.com) → Link
  • ![alt text](image.png) → Image
  • 1. List item → Numbered list
  • * List item or - List item → Bulleted list

Headers:

  • # Header 1 → Header 1
  • ## Header 2 → Header 2
  • ### Header 3 → Header 3
  • #### Header 4 → Header 4

Debbynote's custom markdown syntax:

  • !r [text] → Red highlight for rebuttals
  • !a [text] → White highlight for arguments
  • !wb [text] → Green highlight for worldbuilding
  • !info [text] → Yellow highlight for information

British Parliamentary Debate tokens:

  • !og1 → Opening Government 1st speaker
  • !og2 → Opening Government 2nd speaker
  • !oo1 → Opening Opposition 1st speaker
  • !oo2 → Opening Opposition 2nd speaker
  • !cg1 → Closing Government 1st speaker
  • !cg2 → Closing Government 2nd speaker
  • !co1 → Closing Opposition 1st speaker
  • !co2 → Closing Opposition 2nd speaker
  • !cg1 → Closing Government 1st speaker
  • !cg2 → Closing Government 2nd speaker

Asian Parliamentary Debate tokens:

  • !pm → Prime Minister
  • !lo → Leader of Opposition
  • !dpm → Deputy Prime Minister
  • !dlo → Deputy Leader of Opposition
  • !gw → Government Whip
  • !ow → Opposition Whip
